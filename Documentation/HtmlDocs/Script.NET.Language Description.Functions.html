Script.NET programming language
Language Description->Functions

Functions

function (id1, id2, ... , idn) {
   Statement }

A function creates local context during execution.

Example:

function fac(n){
  if (n==1) return 1;
  else return n*fac(n-1);
}
return fac(5);

//pointer to a function
Func_pointer = fac;
Func_pointer(4); //Call function using pointer

Reserved functions

event – assign event to windows forms control; (Irony Script.NET has other methods for dealing with events.)
eval – evaluates value of an expression;
clear – clears all variables in context;
