Event Handling

Script.NET has limited capabilities of handling .NET events. However it is possible to subscribe function defined in Script on .NET event.

Note: It is possible to subsribe on events having following signature:
public void Event(object sender, T eventArgs) or which is the same EventHandler<T>.

Here is an example:

  function OnClick(s,e)
  {
     MessageBox.Show('Hello');
  }

  button.Click += OnClick();

Please note that by default Script.NET runtime will unsubscribe all event-handlers after execution.
This behavior might be tweaked by using config file.
Please see an Example.
